# 🧠 Sign Language Recognition using Teachable Machine & TensorFlow.js

A simple **web-based Sign Language Recognition** app that uses **TensorFlow.js** and **Teachable Machine** to detect and recognize hand gestures through your **webcam** in real time.

![Sign Language Recognition Screenshot](./screenshot.png)

---

## 🚀 Features

- 🎥 Real-time sign recognition using your webcam  
- 🧩 Integration with **Teachable Machine** image classification model  
- 🪶 Lightweight and fast — runs completely in the browser  
- 🖱️ Simple and modern UI with live status messages  
- 📋 Automatically displays recognized signs as text  

---

## 🧰 Tech Stack

| Technology | Purpose |
|-------------|----------|
| **HTML5 / CSS3 / JavaScript** | Core front-end framework |
| **TensorFlow.js** | Machine learning in the browser |
| **Teachable Machine (Google)** | Model creation and training |
| **Web APIs (MediaDevices)** | Webcam access |
